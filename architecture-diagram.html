Inventory Flow

Stock Ledger Entries

📥 Purchase Entry
PURCHASE_IN Stock Ledger
📤 Challan Delivery
CHALLAN_OUT Stock Ledger
🔄 Return
RETURN_IN Stock Ledger
⚠️
Ledger-Only Stock: Stock is calculated only from stock_ledger entries. There is no direct stock column manipulation.
🔐
Concurrency Safety: PostgreSQL transactions + row-level locking (SELECT ... FOR UPDATE) guarantee safe stock deduction under concurrent requests.

Stock Calculation Model

-- Current stock for a product within an organization
SELECT
  SUM(CASE WHEN type IN ('PURCHASE_IN','RETURN_IN') THEN quantity ELSE 0 END)
  - SUM(CASE WHEN type = 'CHALLAN_OUT' THEN quantity ELSE 0 END)
  AS current_stock
FROM stock_ledger
WHERE organization_id = :active_org_id
  AND product_id = :product_id;

Invoice Structure

Invoice Contains
  • organization_id — Active org
  • hospital_id — Linked hospital
  • doctor_id — Linked doctor
  • Line items & amounts
  • Date & status
Isolation Rules
  • Invoice belongs to active org only
  • Hospital & Doctor are linked references
  • No cross-org invoice visibility
  • Scoped by organization_id
  • Query isolation enforced at app layer
🚫
No Cross-Organization Visibility: An organization can never view, list, or modify invoices belonging to another organization.
